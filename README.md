### pycompressor

Python implementation of PDF set **compressor** (https://arxiv.org/abs/1504.06469).

#### Installation
```
python setup.py install
```
or
```
python setup.py develop
```

####  Running
```
pycompressor -p PDF_NAME -n NB_COMPRESSED_REPLICAS
```
